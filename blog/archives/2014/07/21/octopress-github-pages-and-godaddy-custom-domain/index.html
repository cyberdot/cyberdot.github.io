Octopress, GitHub Pages and GoDaddy Custom Domain

July 21, 2014 octopress, godaddy comments

When I was setting up this blog using Octopress I wanted to configure it to use my cyberdot.co.uk domain name. My domain registrar is GoDaddy.

It turned out to be quite easy to do and here are the steps:

  1. Create a file named CNAME (it has to be in upper case) and save it in your octopress/source directory.

  2. Open CNAME file with your favourite text editor and type in your domain name (in my case it was cyberdot.co.uk)

  3. Don’t forget to commit your file to git and then deploy your octopress blog to GitHub.

  4. Log in to your GoDaddy account, open “Domain details” page and click on “DNS Zone file” tab for your domain

  5. Update A record IP address value to: 192.30.252.153

  6. Also update www ALIAS records to your username.github.io account

  7. Give it some time and lo and behold, you’ve got Octopress + GitHub pages working with custom domain.
